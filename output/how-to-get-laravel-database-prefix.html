How to get laravel database prefix

By Andrew Elkins, Fri 14 June 2013, in category Php, programming

db, laravel, php, prefix

I needed to get the db prefix for a raw insert query I was running. It's an easy function call to get the current database prefix.

DB::getTablePrefix();

Will output the prefix for the current database. This will work in laravel 4.
