Head all files in directory

By Andrew Elkins, Mon 06 June 2016, in category Bash, command line, linux, programming

bash, head, loop

Needed to head all files in a directory.

\#!/usr/bin/env bash  
DEST=/tmp  
for i in /files/to/head/\*  
do  
head -10 "\${i}" &gt; \${DEST}/\$(basename \$i)  
done
